# AU workspace

Use this area for AU-specific ingestion, staging, and transformation work.
- ingest/raw: raw Excel exports (NRM mirrored in ingest/raw/nrm)
- ingest/staging: normalized files ready for staging tables
- ingest/curated: cleaned datasets ready to load
- sql/: migrations, seeds, transforms, views, and RLS policies
- docs/: data dictionary and mapping notes
